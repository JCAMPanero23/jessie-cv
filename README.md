## 📦 Deployment

### GitHub Pages Setup

1. Create new repository on GitHub (name: `jessie-cv`)
2. Clone repository locally or add remote to existing project
3. Commit and push all files
4. Enable GitHub Pages in Settings → Pages → main branch
5. Access at: `https://username.github.io/jessie-cv`

### Local Development

Simply open `index.html` in a browser. No build process required.

## 📝 Updating Content

- **Profile/Contact:** Edit `index.html` header section
- **Projects:** Edit featured projects section in `index.html`
- **Experience:** Edit work experience section
- **Colors:** Edit CSS custom properties in `css/styles.css`
- **Images:** Replace files in `images/portfolio/`
