## Image Specifications

### Resolution:
- **Landscape images:** 1200x800px (3:2 ratio)
- **Portrait images:** 800x1200px (2:3 ratio)
- **Square images:** 1000x1000px (1:1 ratio)

### File Size:
- **Target:** <250KB per image
- **Maximum:** <300KB per image

### Format:
- **JPG** (compressed for web/print)
- Quality: 80-85% (good balance between quality and file size)

### Naming Convention:
Use descriptive names:
- ✅ `wadi-shoka-dining-area.jpg`
- ✅ `desert-rock-balloon-exterior.jpg`
- ✅ `heritage-camp-p1-overview.jpg`
- ❌ `IMG_1234.jpg`

## How to Optimize:

### Using Photoshop:
1. Open image
2. Image → Image Size → Set to 1200x800px (or appropriate size)
3. File → Export → Save for Web (Legacy)
4. Select JPEG, Quality 80
5. Save

### Using Online Tool (Squoosh.app):
1. Go to https://squoosh.app
2. Upload image
3. Resize to 1200x800px
4. Compress: MozJPEG, Quality 80
5. Download

### Using GIMP (Free):
1. Image → Scale Image → 1200x800px
2. File → Export As
3. Select JPEG, Quality 80
4. Export
